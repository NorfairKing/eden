```
$EDEN_ROOT
 |- 001
   |- c
     |- solution.c
 |- lib
   |- c
     |- makefile
     |- primes.c
     |- primes.h

     |- main.cpp
     |- primes_test.cpp
 |- build

```

`EDEN_ROOT` will be set for use in makefiles, for example.
